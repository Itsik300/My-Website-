# My Website

Личная стартовая страница с поиском Google и быстрыми ссылками на полезные сайты.

Опубликованная версия:

```text
https://itsik300.github.io/My-Website-/index.html
```

## Структура

- `index.html` - главная страница со списком ссылок.
- `page2.html` - вторая страница с видео.
- `styles.css` - стили сайта.
- `favicon.png` - иконка сайта.
- `intro-video.mp4` - видео для второй страницы.

Иконки кнопок загружаются как favicon сайтов через Google favicon service, чтобы каждая ссылка использовала узнаваемый оригинальный значок.
Файл `script.js` автоматически добавляет значок к любой ссылке с классом `button` или `login-button`.

## Как редактировать

Чтобы добавить новую ссылку, откройте `index.html` и добавьте новый тег `<a>` в нужный блок `.buttons`.

Минимальный пример:

```html
<a href="https://example.com/" target="_blank" class="button" rel="noopener noreferrer">Example</a>
```

Значок сайта появится автоматически при открытии страницы.

Для внешних ссылок, которые открываются в новой вкладке, используйте:

```html
target="_blank" rel="noopener noreferrer"
```

Это защищает страницу и не дает новой вкладке получить доступ к исходной странице.
